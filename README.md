# 📊 Socio-Economic Analysis of Adult Census Data (Power BI Dashboard)

## 📌 Project Overview

This project presents a **Socio-Economic Analysis of Adult Census Data** using **Microsoft Power BI**.
The dashboard explores demographic and work-related patterns to understand **income distribution, education impact, occupation trends, and working hours** across different population groups.

The goal of this project is to transform raw census data into **interactive visual insights** that help in understanding socio-economic conditions.

# 📂 Dataset Information

**Dataset Name:** Adult Census Income Dataset

The dataset contains demographic, education, occupation, and income information for individuals.

### Important Columns

| Column         | Description                         |
| -------------- | ----------------------------------- |
| Age            | Age of the individual               |
| Education      | Highest education level             |
| Education_Num  | Numeric representation of education |
| Workclass      | Employment category                 |
| Occupation     | Job role                            |
| Hours_Per_Week | Average weekly working hours        |
| Native_Country | Country of origin                   |
| Marital_Status | Marital status                      |
| Race           | Race of the individual              |
| Sex            | Gender                              |
| Capital_Gain   | Capital gains                       |
| Capital_Loss   | Capital losses                      |
| Income         | Income category (≤50K or >50K)      |

# 🔍 Key Insights

### 📚 Education Impact

Higher education levels such as **Doctorate, Professional School, and Master's** show significantly higher percentages of high-income individuals.

### 👔 Occupation Trends

Occupations such as **Executive Managerial and Professional Specialty** show the highest income levels.

### 👨‍👩‍👧 Demographic Factors

Income distribution varies significantly across **gender, race, and marital status**.

### 🕒 Work Patterns

Average working hours vary by **workclass and country**, indicating differences in work culture and employment type.
